# 테스트 생성기

플레이라이트는 즉시 테스트를 생성할 수 있는 기능을 제공하며 이는 테스트를 빠르게 시작할 수 있는 좋은 방법입니다. 테스트 생성기는 두 개의 창을 엽니다. 테스트하려는 웹 사이트와 상호 작용하는 브라우저 창, 그리고 테스트를 기록하고, 복사하고, 지우고, 언어를 변경할 수 있는 플레이라이트 검사기 창입니다.

이곳에서 배우는 것은 다음과 같습니다.

- [코드젠으로 테스트를 생성하는 방법](#코드젠-실행)

## 코드젠 실행

```bash
npx playwright codegen demo.playwright.dev/todomvc
```

코드젠을 실행하고 브라우저에서 작업을 수행합니다. 플레이라이트는 사용자 상호 작용을 위한 코드를 생성합니다. 코드젠은 탄력적인 텍스트 기반 선택기를 생성하려고 시도합니다.

페이지와의 상호 작용이 끝나면 **record** 버튼을 눌러 기록을 중지하고, **copy** 버튼을 사용하여 생성된 코드를 편집기에 복사합니다.

기록를 다시 시작하려면 **clear** 버튼을 사용하여 코드를 지웁니다. 완료되면 플레이라이트 검사기 창을 닫거나 터미널 명령을 중지합니다.

테스트 생성에 대한 자세한 내용이나 안내서는 [코드젠](https://playwright.dev/docs/codegen)을 확인하세요.
